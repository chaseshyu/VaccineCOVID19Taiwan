# VaccineCOVID19Taiwan
因為隨著COVID-19疫情擴散至臺灣社區，參照其他國家後，發現疫苗普及率與疫情趨勢會產生密切相關。因此我稍微整理公開文章與直播中資訊，繪製這個「臺灣COVID-19疫苗接種統計報表」，以利大眾接觸臺灣疫苗資訊。

資料來源：
https://www.cdc.gov.tw/
https://covid-19.nchc.org.tw/
